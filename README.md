
## Prerequisites
* JDK 8 U60 or later
* Maven 3.3.3 or later
* Clang+llvm 4.0.0 or later
* checkstyle-algs4

## Complile like this
``` mvn clean install ```

## Format like this
``` bin/run_clang_fmt.sh ```

## Lint like this
``` ./bin/run_checkstyle.sh ``` 

## Run tests like this
``` mvn test ```
